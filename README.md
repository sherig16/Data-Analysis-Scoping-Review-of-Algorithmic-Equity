# Interactive Tableau Dashboard: Equity in Transformer-Based Healthcare NLP
_Included in a current pre-print publication_

This Tableau dashboard summarizes findings from a scoping review examining **equity-related concerns in transformer-based Natural Language Processing (NLP) models used in healthcare**.  
It synthesizes evidence from **56 peer-reviewed studies**, highlighting patterns in **bias, data representation, and participatory design**.

## Why This Dashboard Matters
- Synthesizes equity-related evidence across 56 peer-reviewed studies
- Enables comparison of bias, data representation, and participatory design trends
- Supports transparent reporting and gap identification in healthcare NLP research

The dashboard was developed using **Tableau Desktop**, based on structured data extracted from a scoping review conducted in accordance with **PRISMA-ScR** and **Arksey & O‚ÄôMalley** methodology.
